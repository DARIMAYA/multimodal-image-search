# Multimodal Image Retrieval and Explanation System

## 📌 Overview
This project implements a multimodal system for aligning images and text using CLIP and BLIP models. It supports cross-modal search and provides explanations for retrieved results.

## 🚀 Features
- Text-to-image search
- Image-to-image retrieval
- Image captioning (BLIP)
- Explanation module

## 🧠 Models
- CLIP (for embeddings)
- BLIP (for captioning)

## 📊 Dataset
Flickr8k dataset

## 🛠️ Tech Stack
- Python
- PyTorch
- Streamlit
- FAISS (optional)
