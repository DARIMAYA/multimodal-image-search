## About the Project

A complete solution for cross-modal image retrieval and understanding, combining modern vision-language models:

- Multimodal embedding pipeline using CLIP  
- Text-to-image and image-to-image search  
- Image captioning with BLIP for semantic understanding  
- Explanation module for interpretable results  
- Interactive web interface for user queries
